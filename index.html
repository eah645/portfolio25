Featured Projects

OIT Hackathon Project

A journaling website that came in first in the Out In Tech Hackathon, built within 25-30 hours. This website allows users to choose a theme (Based on LGBTQ+ identities) and add journal entries. Journal entries are then saved to local storage, and loaded in when the page reloads. Users are able to flip through their journal to see all previous entries.

Nintendo Store

A project built for a web development course final. This is a shopping site, where users can add or remove items from their cart. As users add or remove items, their subtotal changes. Items in the cart are saved to local storage, and will be loaded in on page reloads.

DiaBestics

A Figma prototype created for DiaBestics, an app that can help Type 1 Diabetics manage their levels. This includes features such as counting carbohydrates, setting and tracking goals, schedule tracking, and adjusting notification settings for apps. This serves as a hub to help diabetics.
